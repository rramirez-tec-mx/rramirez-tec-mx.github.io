DATOS GENERALES

Administración de Sistemas Distribuidos
Clave:CS90226
 

 OBJETIVO GENERAL DE LA MATERIA.

Proporcionar al alumno el conocimiento indispensable para analizar el funcionamiento de entornos telemáticos, diseñar la infraestructura para lograr alta escalabilidad, alta confiabilidad, alta disponibilidad, alta seguridad y alto rendimiento. Aprender a realizar una planeación adecuada de la capacidad del entorno

 TEMAS Y SUBTEMAS DEL CURSO

1. Cómputo Distribuido
2. Planeación de Capacidad
3. Administración Linux
4. Análisis de Desempeño de Servidores
5. Administración DNS/DHCP
6. Administración NFS/SMB
7. Administración Base de Datos (MYSQL/POSTGRE)
8. Administración Apache
9. Análisis de desempeño servidor Web
10. Balanceo de Cargas
11. Clusters de Bases de Datos
12. Sistemas de Respaldo a Nivel Dispositivo
13. Alta escalabilidad, alta disponibilidad, alta confiabilidad, alto desempeño

Lista de Proyectos y Actividades

1. Implantación de infraestructura de alto desempeño en entornos Linux (Replicación de servidores de aplicaciones, clusterización de servidores de bases de datos, balanceo de cargas, etc.)
2. Desarrollo de un modelo de planeación de capacidad
3. Implantación de un entorno de manejos de contenidos usando LAMP

LIBROS DE TEXTO
